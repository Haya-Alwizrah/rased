# راصد
مشروع راصد هو نظام ذكي يعتمد على تقنيات الذكاء الاصطناعي لاكتشاف حالات انتحال الهوية في التأمين الطبي. حيث يقوم النظام بتحليل المطالبات الطبية والتقارير المرتبطة بها لاكتشاف اي تناقضات أو انماط غير طبيعيه قد تشير إلى الشخص الذي يستخدم التأمين ليس صاحبه الحقيقي. يعتمد النظام على كل من النموذج اللغوي الكبير "نهى" المقدم من شركة علم بالإضافة إلى تقنية استرجاع المعلومات (RAG)
